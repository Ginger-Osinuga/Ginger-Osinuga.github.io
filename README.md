# Ginger Osinuga – Data Portfolio

Welcome to my data analytics portfolio. This repository showcases projects using Excel, SQL, Tableau, and data visualization techniques to uncover insights and trends.

---

## Student Performance Dashboard (Tableau)
🔗 https://ginger-osinuga.github.io/

This dashboard analyzes student performance across subjects and gender, highlighting differences in average scores and key trends.

### Key Insights
- Reading scores are highest overall, followed by writing, then math.
- Female students outperform males in reading and writing, while males score higher in math.

## International Development Analysis (SQL)
This project analyzes global development data using SQL to identify trends across countries and indicators.
